



类的成员变量
--------------------------------------------------------------------------------------------------------

※ 类的成员变量

   xxx.h 文件
   @Interface Person:NSObject {
       //类的成员变量
	     @public
	     NSString *_name;
	     int _age;
	     float _height;
   }
   @end

   xxx.m 文件
   @implementtation Person
       ....
   @end



 ※ 成员变量的赋值
    Person *p = [[Person alloc] init]; //alloc 为对象分配内存空间  init 进行初始化操作

    p->_name = @"jack";
    p->_age = 19;
    p->_height = 178.5f;
    或者
    (*p)._name = @"jack";
    (*p)._age = 19;
    (*p)._height = 178.5f;

    NSLog(@"name的值是%@",p->_name);


 ※ 成员变量的规范
    1.成员变量是类固有的一些属性，类内使用,如果想要外部直接使用，则应该加 @public，默认为 @protected
    2.成员变量名字规范应该前面加_
    3.成员变量的赋值为p->

属性与成员变量
--------------------------------------------------------------------------------------------------------

※ 成员变量、属性申明
   @Interface People:NSObject{
       //类的成员变量
	     NSString *_name;
	     int _age;
	     float _height;
   }

   //类的属性
   @property NSString *like;

   @end



※ 属性和成员变量对比
    成员变量：类固有的一些属性，类内使用
    属性：是给外部使用的
    类内使用成员变量，类外使用属性

    比如程序员、大脑、嘴、职业
    大脑和嘴自己类内使用，应声明为成员变量
    职业是给类外使用，应声明为属性
