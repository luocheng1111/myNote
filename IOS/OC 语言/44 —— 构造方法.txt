


类的初始化 new 和 alloc init
--------------------------------------------------------------------------------------------------------

 ※ 创建对象
    类名 *指针名 = [类名 new];

    new是1个类方法，其内部
    1.先调用alloc方法，创建对象，分配内存空间
    2.在调用init方法，初始化对象，初始化类中属性的值
    
    同下面一样：
    类名 *指针名 = [[类名 alloc] init];

重写 构造方法
--------------------------------------------------------------------------------------------------------

 ※ 重写init构造方法(规范：按以下格式写)
    
    - (intancetype)init
    {
	self = [super init]; //必须要先对父类的属性初始化
	if(self != nil) //不一定会初始化成功
	{  
	   //自定义初始化属性
	   self.name = @"jack";
	}
	return self;
    }
  


 ※ 自定义构造方法(规范：按以下格式下，方法名必须以initWith开头)
    
    - (intancetype)initWithName:(NSSting *)name andAge:(int)age
    {
	self = [super init]; 
	if(self != nil) 
	{  
	   self.name = name;
	   self.age = age;
	}
	return self;
    }


    Dog *d1 = [[Dog alloc] initWithName:@"小黄" andAge:16];
    
    Dog *d2 = [Dog new];
    [d2 setName:@"小黄"];
    [d2 setAge:16];
