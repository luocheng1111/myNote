

---------------------------------------------------------------------------------------------------
 流程控制基本可包括为三大类
   顺序结构
   循环结构
   选择结构
 
 控制转移
   break;
   continue;
   fallthrough;
   return;
   throw;

---------------------------------------------------------------------------------------------------


循环结构
---------------------------------------------------------------------------------------------------
  for in       for i in 0...10 {...} 
  for          for let i=0；i<100; i++{...}
  while
  repect..while

---------------------------------------------------------------------------------------------------



选择结构
---------------------------------------------------------------------------------------------------
  if..else if..else

  switch
      1.可包含任意数据类型
      2.不用加break；
      3.必须写所有的情况
      4.若多种情况，可写成 case "a", "AA":
      5.若要在default加执行空方法，可写成 break; 或 ();
      6.可进行一些范围判断
        case 0...20 或 0..<20
	case (0, 0) 或 (1, 0)
	case (_, 0) 或 (0, _)
	case (-2...2, -1...3)
      7.若要进入下一个case 加上关键字 fallthrough
      8.case where  case let (x,y) where x == -y :
  
  where 
      case...where
      for case     let i in 1...100 where i%3==0{...}

  gurad 边界检验
      guard money >= price else {
	print(...)
	return
      }

控制转移：跳出循环结构
---------------------------------------------------------------------------------------------------
  break;
  continue;
  fallthrough;
  return;
  throw;
  

  outer: for index in 0..<10{
     if(index==6){
	break outer
     }
  }
