


 Size Class
----------------------------------------------------------------------------------------------------------------------------------------

※ Size Class 介绍
   可以根据不同的屏幕尺寸，呈现不同的界面
   Size Class 不能解决适配问题，它的功能只是将屏幕进行分类，便于管理。
   真正搞适配的苦力还是 AutoLayout 
 

※ Demo见
   \Demo\SizeClass_study01 横屏和竖屏显示不同的控件
   \Demo\SizeClass_study02 iPhone和iPad显示不同的图片，记得要调Deployment Info中的Devices选项，iPhone还是iPad
