※ 新建一个C语言项目
   使用Xcode(C语言编译器, 代替命令行执行cc命令来手动编译、链接、运行)新建一个C项目
   步骤：新建项目 ->macOS ->Application类下的Command Line Tool->输入文件名 选择C语言->一个C程序建立完成->修改运行即可


※ 初识C语言
  #include <stdio.h>  //导入系统的stdio.h头文件 在VC编译器下，必须加这一行，因为VC默认的是C++环境

  int main() //声明一个函数，这个函数名叫做main,这个函数返回值是int类型
  {
      printf("Hello, World!"); //向控制台输出信息。
      return 0; //返回1个数据0
  }
