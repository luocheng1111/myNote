




signed unsigned 有符号和无符号修饰符
-------------------------------------------------------------------------------------------------------------------

※ signed 修饰符
    默认是这个，二进制最高位表示符号位，其他为数据位



※ unsigned 修饰符
    int、short、long、long long 这些变量,全部都是最高位表示符号位，实际上表示的数据少了
    一位，因为最高位用来表示符号位了。
    比如 
      int 占4个字节 31位表示数据 1位表示符号

    而unsigned int 占4个字节 32位来表示数据，
    符号位不用来修饰符号，而是直接参与数据的表示之中，
    那么最大值就翻了一倍，最小值就成了0. 因为没有了正负，


※ unsigned 修饰符 输出表示
    unsigned int --> %u  
    unsigned short --> %hu
    unsigned long --> %lu
    unsigned long long --> %llu
