

C 基础数据类型
-------------------------------------------------------------------------------------------------------------------

 C语言基础数据类型有：int、float、double、char

           int   float   double   char
  字节数     4      4       8       1
  占位符     %d     %f     %lf      %c


-------------------------------------------------------------------------------------------------------------------




整形 int
-------------------------------------------------------------------------------------------------------------------


 int num = 10;
 printf("num = %d", num);

 占4字节
 占位符：%d
 取值范围-2^31 ~ 2^31-1


-------------------------------------------------------------------------------------------------------------------



浮点型 float
-------------------------------------------------------------------------------------------------------------------


 float f = 25.21323f;
 printf("f = %f", f);

 占4字节
 占位符：%f

 小数默认为double类型,如果要为float类型，则应该在后面加f


-------------------------------------------------------------------------------------------------------------------


浮点型 double
-------------------------------------------------------------------------------------------------------------------


 double d = 25.21323;
 printf("d = %lf", d);

 占8字节
 占位符：%lf

 小数默认为double类型


-------------------------------------------------------------------------------------------------------------------



字符型
-------------------------------------------------------------------------------------------------------------------


※ 介绍：
   char类型，占1个字节，占位符：%c, 取值范围-128―127，例如'A'、'9'、'\n'、'\t'
   printf("c = %c", c);

   \b  退格符
   \n  换行符
   \r  回车符
   \t  制表符
   \"  双引号
   \'  单引号
   \\  反斜线


※ 注意点：
   char变量中只能存储单个字符，并且这单个字符是用单引号引起来的
   char变量中无法存储中文(C语言中，1个中文占3个字节，而1个char占1个字节 OC中，1个中文占2个字节)
