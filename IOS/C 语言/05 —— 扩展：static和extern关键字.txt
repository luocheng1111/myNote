


static 和 extern 关键字
-------------------------------------------------------------------------------------------------------------------

※ 介绍
   static 和 extern 可以用来修饰变量和函数
   通常分为三类：
   1). static 和 extern 修饰局部变量
   2). static 和 extern 修饰全局变量
   3). static 和 extern 修饰函数

-------------------------------------------------------------------------------------------------------------------


static和extern 修饰 局部变量
-------------------------------------------------------------------------------------------------------------------

 ※ 介绍
   extern 不能修饰局部变量
   static 修饰局部变量
    1.static修饰的局部变量叫做静态变量，静态变量存储在常量区
    2.即使函数结束，这个变量也不会被回收
    3.这个静态变量只执行一次，下次在调用这个函数时，会自动略过这一行


 ※ 使用
    void test(){
	     static int num = 0
	     num++;
	     print("num = %d\n", num);
    }

    int main(){
      test(); //num = 1
      test(); //num = 2
      test(); //num = 3
      test(); //num = 4
      test(); //num = 5
    }

-------------------------------------------------------------------------------------------------------------------



static和extern 修饰 全局变量
-------------------------------------------------------------------------------------------------------------------

 ※ 介绍
    当我们分模块开发的时候，如果要在模块中声明全局变量
    1).全局变量的声明要写在.h文件中，实现要写在.c文件中
    2).且必须使用static或extern修饰。

    extern修饰：这个模块中的全局变量就可以跨模块访问。
    static修饰：这个模块中的全局变量就只能在当前模块访问。虽仍在别的模块访问static修饰的全局变量不报错，但没有取到真正的值

-------------------------------------------------------------------------------------------------------------------



static 和 extern 修饰函数
-------------------------------------------------------------------------------------------------------------------

 ※ 介绍
    1).如果函数被extern修饰,那么这个函数称为外部函数，它可以被任何源文件中的函数调用。默认是外部函数
    2).如果函数被static修饰,那么这个函数称为内部函数，该函数只能被当前源文件中的其他函数所调用

    如果函数没有写static或者extern，那么函数默认就是extern
