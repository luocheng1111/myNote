


认识 Html
----------------------------------------------------------------------------------------------------------------------------------------

※ Html介绍
   HTML是用来描述网页的一种语言。 他是超文本标记语言(Hyper Text Markup Language)
   HTML提供了一些标签，我们通过这些标签进行排练组合，显示出不同的内容，最后通过网页展示出来
   超文本：超文本是一种组织信息的方式，它通过超链接方法将文本中的文字、图表和其他信息媒体相关联。这些组织信息将分布在不同位置的信息资源用随机的方式进行连接，为人们查找，检索信息提供方便。
   标记预言：使用一套标记标签来描述网页


※ 用法
  <!DOCTYPE html>
  <html>
    <head>
      <title>Title of the document</title>
    </head>

    <body>
      <!--这是一段注释。注释不会在浏览器中显示。-->
      <h1>我的第一个标题</h1>
      <p>我的第一个段落。</p>
    </body>
  </html>


※ Html <!DOCTYPE> 声明
  <!DOCTYPE> 不是 HTML 标签。它为浏览器提供一项信息（声明），即 HTML 是用什么版本编写的。因为HTML 有多个不同的版本，只有明确了HTML版本，浏览器才能完全正确的显示出HTML页面
  例如 HTML5     <!DOCTYPE html>
      HTML4.01  <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"  "http://www.w3.org/TR/html4/loose.dtd">
      XHTML1.0  <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"  "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">


※ Html head元素
  <head> 元素是所有头部元素的容器。<head>内的元素可包含脚本，样式表等等。
  一下标签都可以添加到head部分：<title>、<base>、<link>、<meta>、<script>以及<style>

  <head>
    <meta name="description" content="Free Web tutorials on HTML, CSS, XML" />
    <meta http-equiv="Content-Type" content="text/html; charset=gb2312" />
    <meta http-equiv="Content-Language" content="zh-cn" />

    <title>Title of the document</title>
    <base href="http://www.w3school.com.cn/images/" />
    <base target="_blank" />
    <link rel="stylesheet" type="text/css" href="mystyle.css" />
    <style type="text/css">
       body {background-color:yellow}
       p {color:blue}
    </style>
  </head>

  <title> 定义标题，提供页面被添加到收藏夹时显示的标题
  <base>为页面上的所有链接设定默认地址或默认目标(target)
  <link>定义稳定与外部资源之间的关系，标签最常用于链接样式表
  <style> 为HTML文档定义样式信息
  <script> 用于定义客户端脚本，比如JavaScript
  <meta> 用于提供数据的信息，也叫元数据(metadata), 比如文档的作者，最后修改时间，页面的描述等，元数据不会显示在页面上，但对于机器是刻度的。
         meta标签始终位于head元素中，一些搜索引擎会利用 meta 元素的 name 和 content 属性来索引您的页面。


※ 推荐编辑器
  Notepad (PC) 或 TextEdit (Mac)  或者随意一个文本编辑器


※ Html的后缀
  当您保存 HTML 文件时，既可以使用 .htm 也可以使用 .html 扩展名。两者没有区别，完全根据您的喜好。


※ 注意点
  1.HTML 标签对大小写不敏感：<P> 等同于 <p>。许多网站都使用大写的 HTML 标签。但推荐使用小写
  2.不要忘记结束标签，一般标签都是成对出现的
  3.浏览器会忽略源代码中文字的排版（会省略了多余的空格和换行），比如你文字加了很多空行了或者很多空格，这些浏览器都会忽略， 【https://www.w3school.com.cn/tiy/t.asp?f=html_poem】

※ HTML 元素
  HTML 元素指的是从开始标签（start tag）到结束标签（end tag）的所有代码。
  元素的内容是开始标签与结束标签之间的内容
  例：
  <body>
       <p>This is my first paragraph.</p>
  </body>
  <body> 元素定义了 HTML 文档的主体。这个元素拥有一个开始标签 <body>，以及一个结束标签 </body>。元素内容是另一个 HTML 元素（p 元素）。

  常见的元素
  <h1> - <h6> ：常用来展示标题，自带字体加粗，会自动地在段落的前后添加空行，h1-h6字体大小依次降低。 【https://www.w3school.com.cn/tiy/t.asp?f=html_headers】
  <p> ：用来描述段落，会自动地在段落的前后添加空行  效果展示 ：https://www.w3school.com.cn/tiy/t.asp?f=html_paragraphs1
  <a href="http://www.baidu.com">This is a link</a> ：HTML连接，字体会变蓝，会有下划线，点击字体跳转超链接 【https://www.w3school.com.cn/tiy/t.asp?f=html_paragraphs1】
  <img src="/i/eg_w3school.gif" width="104" height="142" /> 展示图片，设置宽和高   【https://www.w3school.com.cn/tiy/t.asp?f=html_basic_img】
  <hr/>  在页面中创建水平线   【https://www.w3school.com.cn/tiy/t.asp?f=html_hr】
  <br /> 换行   【https://www.w3school.com.cn/tiy/t.asp?f=html_paragraphs】
  <b>This text is bold</b> 文字加粗展示  https://www.w3school.com.cn/tiy/t.asp?f=html_textformatting
  <q> 双引号   https://www.w3school.com.cn/tiy/t.asp?f=html_quotation_q
  <pre> 定义预格式化的文本，文本排版什么样就显示什么样。 https://www.w3school.com.cn/tiy/t.asp?f=html_formatting_codepre



※ HTML 属性
  HTML 标签可以拥有属性。属性提供了有关 HTML 元素的更多的信息。且属性要使用双引号，不过使用单引号也没有问题。
  比如<a>标签的href就是属性
  <a href="http://www.w3school.com.cn">This is a link</a>

  常见的属性
  <h1 align="center"> 居中排列标题 </h1>
  <body bgcolor="yellow"> 设置背景颜色 </body>
  <table border="1"> 设置表格的边框宽度 </table>
  class：规定元素的类名
  id：规定元素的类名
  style：规定元素的样式
  title：规定元素的额外信息


※ HTML 块元素和内敛元素及<div>
  HTML块元素：指元素结束时，会以新的一行开始，通常可用<div>表示，<div>没特定含义，其他的块元素还有<h1>, <p>, <ul>, <table>等...
  HTML内敛元素：内联元素在显示时通常不会以新行开始。可用<span>表示，<span>没特定含义，  比如：<b>, <td>, <a>, <img>

※ Html 表单
  表单由<form>和<input>组成 <form>用于管理和表示他范围内的表单， <input>则由属性的不同，展示不同的样式，可展示输入框、单选、多选、提交按钮等
  例1输入框：
  <form action="action_page.php" >
    First name:<br>
    <input type="text" name="firstname" value="firstname">
    <br>
    Last name:<br>
    <input type="text" name="lastname">
    <br>
    <input type="submit" value="Submit" >
    <input type="button" onclick="alert('Hello World!')" value="Click Me!">
  </form>

  单选框：
  <form>
  <input type="radio" name="sex" checked>Male
  <br>
  <input type="radio" name="sex" >Female
  </form>

  <input>中的属性
   type：有text(输入框)、radio(单选)、checkbox(多选)、submit(提交)、password(密码输入框，字符会被做掩码处理)、button(按钮)等等...
   value：默认值, 如果输入框设置了value，则输入框显示时就会把value值显示在输入框内，可省略不写
   name：用于提交表单时的值，比如上面例1输入框 点击提交后，会请求action_page.php?firstname=Mickey&lastname=Mouse

  <form>中的属性
  全部属性示例
  <form action="action_page.php" method="GET" target="_blank" accept-charset="UTF-8"
    ectype="application/x-www-form-urlencoded" autocomplete="off" novalidate>

  action：提交后，提交表单的地址URL
  method：默认为GET，也可以指定POST
  target：规定 action 属性中地址的目标（默认：_self）
  accept-charset：规定在被提交表单中使用的字符集
  ectype：规定被提交数据的编码（默认：url-encoded）。
  autocomplete：规定浏览器应该自动完成表单（默认：开启）。
  novalidate：规定浏览器不验证表单



※ HTML 类class
  对 HTML 进行分类（设置类），使我们能够为元素的类定义 CSS 样式。
  <html>
    <head>
      <style>
        div.cities {
          background-color:black;
          color:white;
          margin:20px;
          padding:20px;
        }
      </style>
    </head>

    <body>
        <div class="cities">
          <h2>London</h2>
          <p>London is the capital city of England. It is the most populous city in the United Kingdom, with a metropolitan area of over 13 million inhabitants.</p>
          <p>Standing on the River Thames, London has been a major settlement for two millennia, its history going back to its founding by </p>
        </div>
    </body>
  </html>

   注：通常div.cities可以省略写成.cities


※ HTML 的id
  用来定义div的样式，和类class的效果一样，但一般只用于一个div特有的样式 效果演示https://www.w3school.com.cn/tiy/t.asp?f=html_layout_divs
  <html>
    <head>
      <style>
        #cities {
          background-color:black;
          color:white;
          margin:20px;
          padding:20px;
        }
      </style>
    </head>

    <body>
        <div id="cities">
          <h2>London</h2>
          <p>London is the capital city of England. It is the most populous city in the United Kingdom, with a metropolitan area of over 13 million inhabitants.</p>
          <p>Standing on the River Thames, London has been a major settlement for two millennia, its history going back to its founding by </p>
        </div>
    </body>
  </html>

  id和class的区别
  id和class属性是网页中两个通用的属性，它们协同工作使整个页面变得丰富多彩。当我们为一个元素定义样式时。可以使用id，也可以使用class。
  1.书写时，id选择符前面应该加前缀符号‘#’，而class选择符前面应该加前缀符号‘.’。
  2.id属性一般在一个页面中只可以使用一次，而class可以被多次引用。
  3.id作为元素的标签，用于区分不同结构和内容，而class作为一个样式，它可以应用到任何结构和内容上。
