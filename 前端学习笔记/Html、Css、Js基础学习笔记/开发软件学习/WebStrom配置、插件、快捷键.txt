WebStrom 配置/解决运行时卡顿的问题----------------------------------------------------------------------------------------------------------------------------------------1. 调整 webstorm 内存   webstorm 安装目录 > bin > WebStorm.exe.vmoptions。文本编辑器打开，修改第二行和第三行内容。   Mac电脑位置：找到webstorm应用 > 显示包内容 > Contents/bin > webstorm.vmoptions 用文本编辑器打开   第一行：-Xms526m   第二行：-Xmx1024m   这样分配明显好很多，测试发现  -Xms 最大值不能超过1024，否则webstorm将无法打开。2. 把不必要索引的文件进行排除   webstorm 如果同时引入很多个项目也会导致卡顿，我们可以将不需要的项目隐藏起来，和 Eclipse 中 close project 功能一样。   选择项目 > 右键 > Mark Directory As(标记目录为) > Excluded。操作完成后会发现项目“消失了”。   建议取消的索引:node_modules、dist文件夹3. 关闭 node_modules 校验   在 node 项目中存在 node_modules 目录，每次打开 webstrom 时会校验文件，同样也会校验 node_modules 中的内容，这会浪费很多时间。   项目配置 > Editor > File Types > ActionScript > *.as > Ignore files and folders里添加 node_models;4. 取消勾选不常用的插件   webstorm 中可以集成很多插件，这些插件也会影响运行速度，有的插件你可能压根都没听过，更不会使用，可以取消勾选。※ 学习  https://www.jianshu.com/p/08ec229a951f----------------------------------------------------------------------------------------------------------------------------------------WebStrom 快捷键----------------------------------------------------------------------------------------------------------------------------------------   Command+P    快速查找，快速查找项目中的文件及资源   Ctrl+`       (需要自己修改)打开命令行/终端   Command+\    (需要自己添加)拆分文件，代码会非常的长，把当前文件在同一个编辑器拆分开，然后分别滑动到HTML，CSS和脚本部分就可以同时更改了   Command+1    快速打开和关闭侧边栏   ctrl + / : 单行注释   Command+Shift+F      项目中查找某文本   Command+Shift+R      项目中替换查找某文本   Shift+Option+F       代码格式化   Shift+Option+上/下    往上/下复制行   Command+D            复制当前行   Command+Shift+ 左/右  快速选定郑航   Command+X             删除一行   Command+Shift+U       字母转为大小写----------------------------------------------------------------------------------------------------------------------------------------WebStrom 好用的插件----------------------------------------------------------------------------------------------------------------------------------------Atom Material Icons - 属于自动补全插件，它利用深度学习提供智能建议，加速编码过程。Codota - 是一个基于AI的编程插件，它使用机器学习根据具体情境从数百万个代码片段（Java、Javascript、Python等）中实现编码众包。IDE安装时已绑定的插件Weex Language Support - weex文件结构检查，weex语法快速补充weex - 支持自动实时将Weex编译成JScss -
